Simple calendar built with fullCalendar.js. Gives ability to administrate events for multiple users.

== Description ==

Alt Calendar is the simplest way to have calendar on your website.

Every user on your website can:

* add their own calendars
* add new events for their calendars
* edit events with drag&drop or friendly dialogs
* use their own google calendars to have them avaliable on your website


Calendar is provided with basic style for jQuery UI. You can turn it off and make a new one with your own css code.

Plugin supports localization. At this moment it's avaliable in: 

* Polish
* English

Plugin store dates in UTC and show them to user with added timezone.
